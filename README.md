# Análise dos Fatores Socioeconômicos no Desempenho do ENEM: Uma Abordagem com Machine Learning

Este repositório contém todos os arquivos do Trabalho de Conclusão de Curso (TCC) realizado para o **MBA em Ciência de Dados do ICMC-USP** (2026).

O estudo investiga como variáveis socioeconômicas e contextuais influenciam as notas dos estudantes brasileiros no ENEM, utilizando modelos preditivos de alta performance e técnicas de interpretabilidade.
